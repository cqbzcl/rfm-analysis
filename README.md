# 🛒 电商客户RFM分群分析

**业务问题**：如何从54万笔交易中识别高价值客户，实现精准营销？

**核心发现**：633名高价值客户（占15%）贡献60%营收，客均价值是平均水平的4.1倍

## 📊 关键指标

| 指标 | 数值 | 洞察 |
|------|------|------|
| **分析客户数** | 4,319人 | 净消费&gt;0的纯净客户 |
| **高价值客户** | 633人 | RFM≥14分（占15%） |
| **营收贡献** | 293.5万件商品 | 占总营收 **59.8%** |
| **价值密度** | 4.1倍 | 高价值客均是全客户4.1倍 |
| **2-8法则** | 严格符合 | 15%客户贡献60%营收 |

## 🛠️ 技术栈

- **数据处理**：Pandas（清洗+RFM建模）
- **可视化**：Matplotlib/Seaborn（客户金字塔）
- **分析模型**：RFM分群（Recency/Frequency/Monetary）
- **版本控制**：Git/GitHub
- **数据集**：UCI在线零售（54万条交易记录）
